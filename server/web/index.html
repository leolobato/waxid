How Ingestion Works

This system fingerprints audio files so it can identify vinyl records playing in real-time. Upload your music files here to build the fingerprint database.

Supported Formats

  • Audio files: MP3, FLAC, or WAV. Each file is fingerprinted and stored as a track.
  • ZIP archives: A zip file is treated as one album. It can contain audio files and an optional text file with a Discogs URL.

Metadata from Audio Tags

Track metadata is read automatically from audio file tags (ID3 for MP3, Vorbis comments for FLAC). The following tags are used:

  • Album Artist / Artist: Used to identify the album. The albumartist tag is preferred; falls back to artist.
  • Album: The album name. Tracks with the same artist + album are grouped together.
  • Title: The track name. If missing, the filename is used.
  • Track Number: For ordering. Handles "3/12" format.
  • Date / Year: The release year (first 4 digits are used).

If tags are missing, tracks are filed under "Unknown Artist" / "Unknown Album". You can edit this in the album detail view after upload.

Discogs URL

To link an album to its Discogs release page, include a .txt or .md file in your ZIP archive containing the Discogs release URL, e.g.:

https://www.discogs.com/release/12345-Artist-Album-Name

This is optional but useful for reference. The URL is stored with the album and visible in the album detail view.

Tips

  • For best results, use lossless files (FLAC or WAV) ripped from your vinyl records.
  • The fingerprinting works with any audio quality, but higher quality means more reliable matching.
  • You can upload multiple files or ZIPs at once.
  • Cover art is extracted automatically from image files in the ZIP (named cover, front, folder, or artwork) or from embedded audio tags. You can also replace it manually in the album detail view.
  • Side and position info (e.g., "Side A, Track 2") is automatically populated from Discogs when a Discogs link is included in the ZIP (via a .txt or .md file). You can also edit side/position manually in the album detail view.
